Color Gradients

Dan Morriss

Color gradients are a great way to turn make div's background color more interessting. They act as a background image applied using CSS and they look like this.

In this example the color goes from blue to green, from the top to the bottom. The CSS code for this example is appplied to a div with height: 150px; and width; 250px; and looks like this. background-image: linear-gradient(blue, green);

We can do a couple of other things, like change the direction of of gradient by adding in to top right,

background-image: linear-gradient(to top right, blue, green);

We don't need to stick with just two colors, we can add as many as we want, just seperate them with a comma.

background-image: linear-gradient(to top right,black, blue, green, white);

You can use hex or rgb color names too, and specify the direction with degrees:45deg.

These next two lines of code will give the same result.

background-image: linear-gradient(45deg, #1203f9, #007c0d);

CSS Flexible Box 'Flexbox' layout

Megan O'Donohoe

I have been looking into how to create a responsive 'Gallery' page for my static website project. While Grid is an excellent option for large-scale layouts, Flexbox is your best bet for making smooth and responsive, smaller-scale 'Gallery' pages.


An important thing to be aware of, is the more detailed your image is, the longer time it takes to load. You can maintain a lot of the detail while significantly decreasing loading time by using a compressor tool. For the example below I used very small images. For higher resolution images in my project, I will use Optimizilla as the other website I found TinyPNG will only compress images smaller than 5MB free of charge


In addition to the articles from the CSS-Tricks.com website, I also found these two YouTube videos very helpful in creating my Gallery:

  1. How To Create a Responsive Image Grid (Masonry Layout) with HTML and CSS - Flexbox
  2. Responsive Image Gallery with HTML & CSS (Flexbox)

In the Gallery below, three divs with the class of column were created withing a parent div with the class photo-gallery. In CSS, the value of 'flex' enabled the columns to display side-by-side. A media query of 'flex-direction; column:' once the screen size reaches below 500px gives the Gallery a responsive design.

Creating animation using CSS

Georgina Carlisle

The CSS animation properties and their values provide endless opportunities.

You can:

  • Target any HTML element
  • Change the value of any property for the targetted element
  • Control how the annimation runs: speed, repetition, direction, timing
  • Have your annimation complete one or multiple changes

Creating animation purely in HTML and CSS does have one big limitation however. There is only an option to run (although running can be delayed) or pause an animation. This can then only really be controlled by using pseudo-selectors such as :hover (there may be other ways that I haven't discovered). But, add a little java script and the opportunities will increase five-fold.


Things still to be explored:

  • Can multiple elements be targeted with the same animation?
  • How well does it work to change multiple properties at the same time?
  • Using the cubic-bezier values for the annimation-timing-function.
  • Using the animation shorthand.
  • And or course ... adding in some javascript!

The following demonstration has been created purely using HTML and CSS. The balloon is constantly 'running'. The rocket annimation is set to 'pause' and will only run when clicked AND the click held. The treasure island is also set to 'pause' and will run when hovered over.
