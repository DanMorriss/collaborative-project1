Color Gradients

Dan Morriss

I have recently been looking into background gradients. These act as a background image applied using CSS and they look like this.

In this example the color goes from blue to green, from the top to the bottom. the CSS code for this looks like this. background-image: linear-gradient(blue, green);

We can do a few more interesting things, like change the direction of of gradient by adding in to top right,

background-image: linear-gradient(to top right, blue, green);

We don't need to stick with one color, we can add as many as we want, just seperate them with a comma.

background-image: linear-gradient(to top right,black, blue, green, white);

CSS Flexible Box 'Flexbox' layout

Megan O'Donohoe

I have been looking into how to create a responsive 'Gallery' page for my static website project. While Grid is an excellent option for large-scale layouts, Flexbox is your best bet for making smooth and responsive, smaller-scale 'Gallery' pages.


An important thing to be aware of, is the more detailed your image is, the longer time it takes to load. You can maintain a lot of the detail while significantly decreasing loading time by using a compressor tool. For the example below I used very small images. For higher resolution images in my project, I will use Optimizilla as the other website I found TinyPNG will only compress images smaller than 5MB free of charge


In addition to the articles from the CSS-Tricks.com website, I also found these two YouTube videos very helpful in creating my Gallery:

  1. How To Create a Responsive Image Grid (Masonry Layout) with HTML and CSS - Flexbox
  2. Responsive Image Gallery with HTML & CSS (Flexbox)

In the Gallery below, three divs with the class of column were created withing a parent div with the class photo-gallery. In CSS, the value of 'flex' enabled the columns to display side-by-side. A media query of 'flex-direction; column:' once the screen size reaches below 500px gives the Gallery a responsive design.

Creating annimation using CSS

Annimation using CSS can be activated on:

  • page loading
  • element hover
  • element click
  • ... and more

Annimation using CSS can:

  • change from one style or set of styles to another
  • change as fast or slow as you dictate
  • Have a one step or multiple step change
