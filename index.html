Color Gradients

Color gradients are a great way to turn make div's background color more interessting. They act as a background image applied using CSS and they look like this.

In this example the color goes from blue to green, from the top to the bottom. The CSS code for this example is appplied to a div with height: 150px; and width; 250px; and looks like this. background-image: linear-gradient(blue, green);

We can do a couple of other things, like change the direction of of gradient by adding in to top right,

background-image: linear-gradient(to top right, blue, green);

We don't need to stick with just two colors, we can add as many as we want, just seperate them with a comma.

background-image: linear-gradient(to top right,black, blue, green, white);

You can use hex or rgb color names too, and specify the direction with degrees:45deg.

These next two lines of code will give the same result.

background-image: linear-gradient(45deg, #1203f9, #007c0d);

Creating animation using CSS

The CSS animation properties and their values provide endless opportunities.

You can:

  • Target any HTML element
  • Change the value of any property for the targetted element
  • Control how the annimation runs: speed, repetition, direction, timing
  • Have your annimation complete one or multiple changes

Creating animation purely in HTML and CSS does have one big limitation however. There is only an option to run (although running can be delayed) or pause an annimation. This can then only really be controlled by using pseudo-selectors such as :hover (there may be other ways that I haven't discovered). But, add a little java script and the opportunities will increase five-fold.

Things still to be explored:

  • Can multiple elements be targeted with the same animation?
  • How well does it work to change multiple properties at the same time?
  • Using the cubic-bezier values for the annimation-timing-function.
  • Using the animation shorthand.
  • And or course ... adding in some javascript!
